# Shared UI Components

Dieser Ordner enthält alle geteilten UI-Komponenten (Header, ChatPanels), 
die sowohl von der Legacy-App (`variants/legacy`) als auch von zukünftigen 
Stitch-generierten Designs (`variants/stitch-v1`, etc.) genutzt werden.

Zukünftige KI-generierte React-Komponenten werden einfach in ihren 
jeweiligen `variants/*` Ordner geschoben und können diese Elemente 
nahtlos importieren.
